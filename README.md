#Mango App
Бекенд web server приложение для социальной сети [ месенджера ]. 

Стек технологий:
- FastAPI
-  SQLAlchemy
-  Websockets 
-  OAuth2
-  JWT Authentication
-  Sentry SDK
-  Docker
-  Chat GPT [Open AI]
-  Open API
-  Swagger
